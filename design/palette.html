The background is specified in linear light space in the wgpu clear op. When the surface format is sRGB (typical on macOS), the GPU converts to display-referred on output — so the visual result is approximately #3F3F59, a dark blue-slate. Use the linear tuple (0.05, 0.05, 0.10) as the authoritative value when writing GPU code; use the hex approximation only for CSS reference.
